CSS 指层叠样式表 (Cascading Style Sheets)

CSS 由两部分构成：1.选择器selector，2.声明property: value

如何写一条CSS:1.确定选择器selector,2.写声明property: value

        /* 示例 */
        pre{
            background-color: #ddd;
            padding: 10px 0;
            margin: 0;
        }
编写CSS的3种方式
        <head>
            
            <style type="text/css">
                hr {color: sienna;}
                p {margin-left: 20px;}
                body {background-image: url("images/back40.gif");}
            </style>
            
            <link rel="stylesheet" type="text/css" href="mystyle.css" />
        </head>
        <body>
            <div style="color:#f00;"></div>
        </body>
